[html.md][] is a pure JavaScript library for converting [HTML][] in to valid
[Markdown][].

## Standard Usage

``` javascript
md(html[, options])
```

## Miscellaneous

``` javascript
md.noConflict()
md.VERSION
```

currently in development you can do so here;
